DataTables and Bootstrap 3

Minimal setup of DataTables with Bootstrap 3. It uses the following versions:

  • Bootstrap dist 3.0.2
  • DataTables 1.9.4
  • Bootstrap 3 integration with DataTables (including the images in ../images/)
  • Bootstrap pagination plugin, with some changes.

All the files needed for this example, from above distribution archives and from above discussions in forums, including this very example page, can be downloaded as an archive file (including the version of jQuery included with DataTables, jQuery 1.8.2, so the example is really self-contained).
